__________________________________________________________________________________________________________________________________________________________________________________________________________________________________
1) Download SVN server
__________________________________________________________________________________________________________________________________________________________________________________________________________________________________
2) Set environment

SVN_EDITOR = C:\Windows\Notepad.exe
SVN_HOME = <folder where you installed SVN>

-> you can skip this, if you run all the commands from the SVN_HOME\bin folder
PATH = %PATH%;%SVN_HOME%;%SVN_HOME%\bin           

__________________________________________________________________________________________________________________________________________________________________________________________________________________________________
3) Create a folder for the repository

C:\SvnRepo

__________________________________________________________________________________________________________________________________________________________________________________________________________________________________
4) Run cmd:

svnadmin create "C:\SvnRepo"

__________________________________________________________________________________________________________________________________________________________________________________________________________________________________
5) Go to C:\SvnRepo\conf -> edit svnserve.conf

uncomment anon-access=read
          auth-write=write
          password-db=passwd
          
Edit passwd file -> add users & passwords

__________________________________________________________________________________________________________________________________________________________________________________________________________________________________
6) Run cmd:

sc create svn.local binpath= "<svn_bin_dir>\svnserve.exe --service --root C:\SvnRepo" displayname= "Subversion" depend= Tcpip

!!! Make sure you leave a space after each "=": binpath=<space>"....
