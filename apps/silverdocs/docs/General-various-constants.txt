Various computer-related constants
-----------------------------------------------------------------------------------------------------------------
-----------------------------------------------------------------------------------------------------------------
__________________________________________________________________________________________________________________________________________________________________________________________________________________________________
Line delimiters:

Windows - \r\n
Unix    - \n
Mac     - \r

__________________________________________________________________________________________________________________________________________________________________________________________________________________________________
Java remote debugging: setting the "runjdwp" variable:

Env variable name : JAVA_OPTS
Env variable value: -Xrunjdwp:transport=dt_socket,server=y,suspend=n,address=9225

__________________________________________________________________________________________________________________________________________________________________________________________________________________________________
Set trusted keystore for Java (i.e. for SSL tests): 

Env variable name : JAVA_OPTS
Env variable value: -Djavax.net.ssl.trustStore=amadeus_keystore

__________________________________________________________________________________________________________________________________________________________________________________________________________________________________
Configure Java programs to access Internet through a proxy:

java -DproxySet=true -DproxyHost=100.200.100.200 -DproxyPort=1234 MyClass

__________________________________________________________________________________________________________________________________________________________________________________________________________________________________
Configure BEA Weblogic 8.1 for debugging with Eclipse:

Add the following line in startWebLogic.cmd, in .../domains/mydomain

set JAVA_OPTIONS=-Xdebug -Xnoagent -Xrunjdwp:transport=dt_socket,server=y,suspend=n,address=9225
